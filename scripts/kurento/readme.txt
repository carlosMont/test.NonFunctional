Contents of the package:

- kms_monitoring:
Contents a Java program that gather all the metrics used. 
	 bin: Contents a .jar file, which is the binary to execute and gather the metrics, a .bat to execute it, and a /lib folder with all the libraries needed
	 source: Contents a maven java project to generate .jar binary.

- kurento-hello-world: 
Maven project of Loopback example from Kurento's documentation page. It has been used as the test case for the performance testing.

- selenium:
Contents a selenium scripts, used in the test.
	 KurentoClientLB.bat: executable for the test
	 KurentoClientLB.jar: binary selenium scripts
	 workspace.zip: full workspace of the selenium script
